# Catalog
This is the song-tracker. 
Firstly it was adapted only for music, but then I decided to improve it to make it for more.
When you start the program, firstly you need to choose whether you want to add movie or song.This catalog has an opportunity to add (using Scanner through terminal), delete by artist or director, sort by music duration/movie by year, and find by artist/director. 
For now it works with your own input, but further I want to make a database, to work properly. 
This program was written on Java, using basic concepts such as Encapsulation, Generic Interface, Scanner.
To run the program, simply clone the repo and run the Main.java
