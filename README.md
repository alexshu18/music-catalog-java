# Music Catalog
This is the song-tracker. 
Firstly it was adapted only for music, but then I decided to improve it to make it for more.
When you start the program, this music catalog(for now) has an opportunity to add track(using Scanner through terminal), delete track by artist, sort by music duration, and find by artist. 
For now it works with your own input, but further I want to make a database, to work properly. 
This program was written on Java, using basic concepts such as Encapsulation, Interface, Scanner.
To run the program, simply clone the repo and run the Main.java
